# Lab-in-A-Box Website

A premium, interactive single-product website for the Lab-in-A-Box electronics kit built with Next.js.

## Project Setup

### Prerequisites

- Node.js installed on your machine (v18 or higher recommended).

### Installation

If you haven't already, install the dependencies:

```bash
npm install
```

### Running the Project

To start the development server:

```bash
npm run dev
```

The site will be available at `http://localhost:3000`.

### Build for Production

```bash
npm run build
npm start
```

## Technology Stack

- **Framework**: Next.js 14 (App Router)
- **UI Library**: React 18
- **Styling**: CSS Modules

## Features

- **Hero Section**: 3D parallax product display.
- **Interactive Gallery**: Auto-rotating image carousel.
- **Animated Specs**: Technical specifications with scroll animations.
- **Order Form**: Validation and success states with multiple payment methods.
